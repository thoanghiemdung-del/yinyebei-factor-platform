# 银叶杯提交成果包

## 建议阅读顺序

1. `银叶杯最终研究报告.pdf`
2. `因子明细\最终10因子经济含义.md`
3. `因子明细\最终10因子明细.tsv`
4. `审计与验证\中文交付审计.json`

## 核心结论

- 最终冻结因子：10 个
- 实测样本外 Sharpe：最低 10.722，最高 16.247
- 任意两因子的全窗口日度 PnL 绝对 Pearson 相关性：均小于 0.5
- 最大两两相关性：0.4980399457
- 新增受控扩展实验：950 条真实记录
- 创新实验阶段：11 个
- 中文论文：15 页

最终因子主要覆盖三类经济机制：

1. 慢速信息扩散驱动的趋势延续；
2. 订单流冲击后的短期价格修复；
3. 流动性、容量和拥挤度约束下的可交易性筛选。

## 文件结构

- `银叶杯最终研究报告.pdf`：中文最终论文
- `银叶杯最终研究报告.tex`：LaTeX 源文件
- `因子明细\`：最终 10 因子的冻结清单、中文明细和详细经济解释
- `审计与验证\`：中文交付审计、创新阶段统计、顺序分段审计和留出集预注册
- `图表\`：论文中的四张中文图表
- `提交说明.md`：原始中文交付索引

## 证据边界

2023 年研究阶段样本外窗口已经被自适应检查。当前结果是可审计的内部研究成果，不应解释为未知未来收益保证。

正式对外提交性能主张前，仍需执行未触碰留出集验证，并补充交易成本、容量、涨跌停、T+1、滚动相关性和多重检验审计。

本成果包未提交任何外部 Alpha。

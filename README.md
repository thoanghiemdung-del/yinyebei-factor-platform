# 银叶杯量化因子研究平台

本仓库是银叶杯因子项目的平台源码整理版，保留平台、回测、表达式解析与因子研究相关代码，不包含比赛原始分钟数据、回测数据库、缓存和大体积中间产物。

## 平台功能

- 因子表达式输入与解析：支持将价量、分钟派生变量和组合算子解析为可回测因子。
- 数据管线：对交易日、股票池、标签和因子矩阵进行统一对齐。
- 回测引擎：支持 Top10% 组合收益、Pearson IC、年化超额收益、Sharpe、最大回撤、换手率等指标计算。
- 因子历史管理：使用 SQLite 保存 Alpha 历史、回测指标、净值曲线和社区讨论记录。
- 可视化界面：Flask Web 平台提供登录、表达式回测、指标看板、相关性检查、数据字段说明和学习页面。
- 平台化交付：将叶子因子生成、风格组合、终极因子训练、报告生成和提交包校验串联为可复用工作流。

## 目录结构

```text
.
├── backtest_platform/       # Flask 平台、页面模板、静态资源和研究脚本
├── model/                   # 数据管线、回测引擎、表达式解析和因子库
├── docs/                    # 平台说明与交付说明
├── requirements.txt         # Python 依赖
└── .gitignore               # 排除数据、缓存、日志和数据库
```

## 运行方式

安装依赖：

```bash
pip install -r requirements.txt
```

启动平台：

```bash
cd backtest_platform
python app.py
```

默认本地地址通常为 `http://127.0.0.1:5000`。平台需要比赛数据文件才能完整回测；本仓库只上传源码，不上传原始数据和中间数据库。

## 与最终因子的关系

最终提交因子 `ultimate_lgb` 来自该平台的一套研究流程：先生成多类叶子因子，再在流动性、价格形态、动量、容量、成交活跃度、日内微观结构等风格内组合，最后使用正则化 LightGBM 将风格因子合成为单一终极因子。平台是本次项目的重要中间成果，也是后续继续扩展因子库和回测流程的基础。

# 官方初赛提交逻辑

## 最终因子

最终提交只有一个因子：`ultimate_lgb`。

- 行：交易日，2020-01-02 至 2023-12-29，共 970 行。
- 列：股票代码，共 5515 只。
- 值：每日横截面 winsorize/标准化后的最终得分。

## 研究路径

1. 从日频和分钟源数据构造叶子因子。
2. 按经济风格组合叶子因子，风格包括但不限于：
   流动性/换手、价格形态、多周期动量、容量约束、成交活跃度、日内微观结构。
3. 在 2020-2022 IS 内比较不同组合方式，确定风格因子。
4. 使用 LightGBM 将风格因子非线性合成为 `ultimate_lgb`。
5. 仅最终报告 2023 OOS 结果，并冻结 `因子值.xlsx`。

## 为什么提交脚本读取 Excel

比赛提交包需要稳定、轻量、可被评委直接读取。`银叶杯.py` 因此只做最终冻结矩阵的加载与校验；
从源数据重算的完整逻辑在平台源码和模型代码中保留，用于审计、复现和继续研究。
